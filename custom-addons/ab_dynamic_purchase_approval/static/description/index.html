Purchase Order Dynamic Approval

This module it's able to set your dynamic approval for Purchase Order in Companies. You can set the approval by many Groups or Users and depending with Minimal Order Amount.

Setting Dynamic Approval On Companies:


User friendly with many validation:

  • 1. User can't set first approval amount order greater than next approval.
  • 2. User can't set the first approval amount order lower than previous approval.
  • 3. User can't change approval type while line it's not empty.



Sample Case On Purchase Order:

In this case, i'll create Purchase Document (PO00004) by "Purchase User" with Rp.2.300.000 order amount. And this document can be approve by "Administrator and Marc Angelo" according to the example image above.


When the document it's confirmed, he can't see every single button on this document.

User "Administrator" it's the last approval. When the button "Approve" clicked by the last approval, the system automatically generate the document to Purchase Order.


Refuse Order:

The approval it's able to refuse the document by click "Refuse" button.
